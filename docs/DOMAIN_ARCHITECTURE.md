## Rollback Plan

If something goes wrong, you can quickly rollback:

### Option 1: Remove Custom Domain from CloudFront
1. Edit CloudFront distribution
2. Remove alternate domain names
3. Users can still access via CloudFront URL

### Option 2: Change DNS Back
1. Remove/modify DNS records in Porkbun
2. Point to old infrastructure (if any)

### Option 3: Use CloudFront URL Temporarily
- Share CloudFront URL directly
- No DNS changes needed
- Works immediately
